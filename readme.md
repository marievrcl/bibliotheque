# Projet ALGO C ING3 
## RÉSUMÉ – Quel est le contexte et la problématique du projet ?
Le projet, réalisé en 7 semaines dans le cadre des cours de bases de données et de programmation en C, vise à mettre en pratique les compétences acquises et à développer l’autonomie. La problématique consiste à automatiser la gestion d’une bibliothèque numérique afin de suivre efficacement les livres, les utilisateurs et les emprunts, tout en assurant la sauvegarde et le chargement des données.

## Quels sont les objectifs techniques ?
Les objectifs techniques incluent : la conception d’un modèle relationnel (MCD/MLD), le développement d’une application C modulaire, la manipulation de structures, pointeurs et fichiers, et la gestion complète des entités du système. Parmi les fonctionnalités avancées, l’équipe a choisi d’implémenter des statistiques sur les emprunts. Les outils utilisés sont CLion pour le développement et MySQL pour la base de données.
